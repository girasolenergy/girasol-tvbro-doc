# プロモーションコード利用ガイド

## 手順

### プロモーションコードを入手する

プロモーションコードを入手します。

プロモーションコードは `123456789ABCDEFGHIJKLMN` のような英大文字と数字で構成された23文字のコードです。

### プロモーションコードを入力する

まず、Google Playの右上のアカウントのアイコンからアカウントの管理画面に遷移します。

<img alt="Google Playのメイン画面のスクリーンショット" src="image.png" width="400">

---

アカウントの管理画面上で「お支払いと定期購入」を選択します。

<img alt="アカウントの管理画面のスクリーンショット" src="image-1.png" width="400">

---

「コードの利用」を選択します。

<img alt="「お支払いと定期購入」画面のスクリーンショット" src="image-2.png" width="400">

---

プロモーションコードを入力すると、KanbanBroが無料で入手可能になります。

<img alt="プロモーションコードを入力し終えた状態のスクリーンショット" src="image-3.png" width="400">
