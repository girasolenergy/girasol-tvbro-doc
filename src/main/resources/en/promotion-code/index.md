# Promotion Code Usage Guide

## Steps

### Get a promotion code

Obtain a promotion code.

A promotion code is a 23-character string of uppercase letters and digits, such as `123456789ABCDEFGHIJKLMN`.

### Enter the promotion code

First, in Google Play, tap the account icon in the top right to open the account management screen.

<img alt="Screenshot of the Google Play home screen" src="image.png" width="400">

---

On the account management screen, select "Payments & subscriptions".

<img alt="Screenshot of the account management screen" src="image-1.png" width="400">

---

Select "Redeem code".

<img alt="Screenshot of the Payments & subscriptions screen" src="image-2.png" width="400">

---

After entering the promotion code, KanbanBro will be available at no charge.

<img alt="Screenshot after entering the promotion code" src="image-3.png" width="400">
